Introduction

The Crowd Navigation system aims to allow visually impaired users gain assistance in navigation (and other tasks) using crowdsourced assistance. The system is under development and this site is part of the testing grounds. Please note that the video streaming system uses WebRTC, which is currently only supported by the Chrome and Firefox browsers. However, WebRTC is currently being drafted as a W3C standard, so more browsers will likely support it soon. Also, note that the system uses Google App Engine and Google Accounts to manage users. Thus, the system will ask for basic Google Account permissions to manage user data.

Static Demo

A conceptual demo can be found by clicking the demo button. Rather than stream from a real person, there is a looped video being played. If multiple users log into the demo page, their directions (given by the arrow keys) will be aggregated and displayed.

VR Demo

A virtual reality demo can be found by clicking the demo button. Rather than stream from a real person, there is a virtual reality environment simulating a real user. If multiple users log into the demo page, their directions (given by the arrow keys) will be aggregated and used to direct the virtual avatar.

Create Room

The create room button will create a live stream from your browser. This can be run from a Chrome or Firefox browser on an Android mobile device. If using Firefox, the rear camera on an Android device can be selected on the page load. Audio and text feedback is provided to the user.

Development

The development button leads to the development version of the system. As this is being used in the testing and development, this page could be in any state.

Open Streams

The open streams provide a list of live streams that are available.
